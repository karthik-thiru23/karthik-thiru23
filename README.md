<details>
<summary><b>⚡ QuorumEval — Correlation-Aware LLM Jury</b></summary>
<br>
LLM evaluation framework that reduces scoring bias by detecting and down-weighting correlated judge agreement. Running on AMD MI300X for the AMD Developer Hackathon ACT II. The insight: consensus hides blind spots; disagreement is the signal.
<br><br>
<code>AMD MI300X</code> <code>Python</code> <code>LLM Evaluation</code> <code>Statistical Methods</code>
</details>

<details>
<summary><b>🤖 Friday AI — Spec-Driven Agentic Coding Assistant</b></summary>

Takes a Jira-style spec and drives code generation across Angular, iOS, Android, and Node.js stacks with minimal human in the loop. Built on Claude Code with a spec-first architecture that keeps the agent grounded.
<br><br>
<code>Claude Code</code> <code>Angular</code> <code>Node.js</code> <code>Agentic AI</code>
</details>
