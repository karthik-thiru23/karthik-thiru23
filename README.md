## Selected personal projects

**ASTRA Hub** — Engineering intelligence platform that connects a Neo4j knowledge graph, Qdrant vector search, and a FastAPI orchestration layer into a multi-agent system for codebase reasoning. Built as an internal tool to make large engineering orgs legible.

**QuorumEval** — LLM evaluation framework using a correlation-aware jury approach to reduce scoring bias. Running on AMD MI300X for the AMD Developer Hackathon ACT II. Explores how disagreement between judges surfaces blind spots that consensus hides.

**Friday AI** — Spec-driven agentic coding assistant. Takes a Jira-style spec and drives code generation across Angular, iOS, Android, and Node.js stacks with minimal human in the loop.

**Kino** — Local-first desktop movie tracker. Tauri · React · SQLite. Built to scratch a personal itch; ended up as a clean example of shipping a native desktop app without Electron overhead.
